# 🧬 Advanced Cancer Prediction System

A comprehensive multi-cancer risk assessment platform powered by ensemble machine learning models and an interactive graphical user interface.

---

## 📌 Overview

The **Advanced Cancer Prediction System** is a Python-based application designed to predict the risk of multiple cancer types using clinical and imaging data. It combines multiple machine learning models into an ensemble framework and presents results through a user-friendly GUI built with Tkinter.

This project bridges the gap between complex healthcare analytics and accessible clinical tools, enabling early detection and informed decision-making.

---

## 🎯 Features

- 🔍 Multi-Cancer Prediction (28 cancer types)
- 🧠 Ensemble Machine Learning Models
- 📊 Real-time Risk Assessment with Confidence Scores
- 🖥️ Interactive GUI (Tkinter-based)
- 🧾 Prediction History Tracking
- 📈 Visual Analytics Dashboard
- 🔄 Dual Input Modes:
  - Clinical Data Mode
  - Imaging Features Mode

---

## 🛠️ Tech Stack

- **Language:** Python  
- **Machine Learning:** scikit-learn  
- **Data Processing:** pandas, numpy  
- **Visualization:** matplotlib, seaborn  
- **GUI:** tkinter

## ⚙️ Installation

1. Clone the repository
```bash
git clone https://github.com/your-username/cancer-prediction-system.git
cd cancer-prediction-system
```

2. Install required dependencies
```bash
pip install -r requirements.txt
```

If requirements.txt is not provided:
```bash
pip install numpy pandas scikit-learn matplotlib seaborn
```

---

## ▶️ Usage

Run the application:
```bash
python main.py
```
Replace main.py with your actual file name if different.
